# Retail Business Analysis - SQL

## Project Overview
This project analyzes retail sales data using SQL to generate meaningful business insights. It includes data exploration, sales analysis, customer segmentation, and profitability analysis using SQL queries.

## Objectives
- Analyze retail sales performance
- Identify top-performing products and categories
- Analyze regional sales and profit
- Evaluate customer segments
- Generate business insights using SQL

## Tools Used
- MySQL
- SQL

## Dataset
Superstore Retail Sales Dataset

## Analysis Performed
- Total Sales
- Total Profit
- Total Orders
- Sales by Category
- Profit by Category
- Sales by Region
- Top 10 Customers
- Top 10 Products
- Monthly Sales Trend

## Key Insights
- Technology generated the highest sales.
- West region recorded the highest revenue.
- Consumer segment contributed the largest share of sales.
